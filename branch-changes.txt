================================================================================
SUMMARY
================================================================================

This branch replaces string constants throughout the codebase with type-safe
enums using Strum derives (AsRefStr, Display, EnumString, IntoStaticStr).
This provides compile-time safety, better IDE support, and eliminates typo bugs.
