# Standard Library Base58 Functions {{ id: "std-functions" }}


## encode_base58 {{ tag: 'function' }}

<Row>
  <Col>

`encode_base58` encodes a buffer or string as a base58 string.

### Inputs

<Properties>


  <Property name="value" required="required" type="buffer | string | addon">
    The buffer or string to encode. Strings starting with '0x' are decoded as hex; otherwise, raw UTF-8 bytes are used.
  </Property>



</Properties>

### Output

<Properties>
  <Property name="value" type="string">
    The input, encoded as a base58 string.
  </Property>
</Properties>


<CodeGroup title="Example using encode_base58">


```hcl {{ title: 'main.tx' }}
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF

```

## decode_base58 {{ tag: 'function' }}

<Row>
  <Col>

`decode_base58` decodes a base58 encoded string and returns the result as a buffer.

### Inputs

<Properties>


  <Property name="base58_string" required="required" type="string">
    The base58 string to decode.
  </Property>



</Properties>

### Output

<Properties>
  <Property name="value" type="buffer">
    The decoded base58 string, as a buffer.
  </Property>
</Properties>


<CodeGroup title="Example using decode_base58">


```hcl {{ title: 'main.tx' }}
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8

```
