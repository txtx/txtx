## account {{ tag: 'function' }}

<Row>
  <Col>

`svm::account` encodes a public key in to an account meta object for a program instruction call.

### Inputs

<Properties>


  <Property name="public_key">
    The on-chain address of an account.
  </Property>


  <Property name="is_signer">
    Specify if the account is required as a signer on the transaction.
  </Property>


  <Property name="is_writable">
    Specify if the account data will be modified.
  </Property>


</Properties>

### Output

<Properties>
  <Property name="value">
    The account meta object.
  </Property>
</Properties>


<CodeGroup title="Example using account">



## find_pda {{ tag: 'function' }}

<Row>
  <Col>

`svm::find_pda` finds a valid pda using the provided program id and seeds.

### Inputs

<Properties>


  <Property name="program_id">
    The address of the program the PDA is derived from.
  </Property>


  <Property name="seeds">
    An optional array of seeds that will be used to derive the PDA. A maximum of 16 seeds can be used, and each seed can have a maximum length of 32 bytes.
  </Property>


</Properties>

### Output

<Properties>
  <Property name="value">
    An object containing the PDA address and associated bump seed.
  </Property>
</Properties>


<CodeGroup title="Example using find_pda">


```hcl {{ title: 'main.tx' }}
variable "pda" {
    value = svm::find_pda("3bv3j4GvMPjvvBX9QdoX27pVoWhDSXpwKZipFF1QiVr6", ["data"])
}
output "pda" {
    value = std::encode_base58(variable.pda.pda)
}
output "bump" {
    value = variable.pda.bump_seed
}
// > pda: 4amHoWMBgLkPfM8Nq9ZP33Liq9FCuqrLoU1feejkdsUJ
// > bump: 252

```

  <Col>

### Inputs

<Properties>


  <Property name="funding_address">
    The address of the funding account.
  </Property>


  <Property name="wallet_address">
    The address of the wallet to compute the associated token account for.
  </Property>


  <Property name="token_mint_address">
    The address of the token mint used to compute the token account.
  </Property>


  <Property name="token_program_id">
    The address of the token program used to compute the token account.
  </Property>


</Properties>

### Output

<Properties>
  <Property name="value">
    The serialized instruction bytes.
  </Property>
</Properties>


<CodeGroup title="Example using create_token_account_instruction">


```hcl {{ title: 'main.tx' }}

    signers = [signer.caller]

        value = svm::create_token_account_instruction(
```
