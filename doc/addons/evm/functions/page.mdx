## get_contract_from_foundry_project {{ tag: 'function' }}

<Row>
  <Col>

`evm::get_contract_from_foundry_project` retrieves the compiled contract artifacts for a contract in a Foundry project.

### Inputs

<Properties>


  <Property name="contract_name" required="required" type="string">
    The name of the contract being deployed.
  </Property>


  <Property name="contract_filename" required="optional" type="string">
    The `.sol` file that the contract is located in. Defaults to `<ContractName>.sol`.
  </Property>


  <Property name="foundry_manifest_path" required="optional" type="string">
    The location of the Foundry.toml. Defaults to `./foundry.toml`.
  </Property>


  <Property name="foundry_profile" required="optional" type="string">
    The foundry profile that should be used to find the compiled output. Defaults to `default`.
  </Property>



</Properties>

### Output

<Properties>
  <Property name="value" type="object">
    Coming Soon
  </Property>
</Properties>

    value = evm::get_contract_from_foundry_project("MyContract")
