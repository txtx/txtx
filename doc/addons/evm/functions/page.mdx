## address {{ tag: 'function' }}

<Row>
  <Col>

`evm::address` creates a valid Ethereum address from the input string.

### Inputs

<Properties>


  <Property name="address_string">
    An Ethereum address string.
  </Property>


</Properties>

### Output

<Properties>
  <Property name="value">
    The input string as an Ethereum address.
  </Property>
</Properties>

## bytes32 {{ tag: 'function' }}

<Row>
  <Col>

`evm::bytes32` encodes a hex string as a 32-byte buffer.

### Inputs

<Properties>


  <Property name="input">
    A 32-byte hexadecimal string.
  </Property>


</Properties>

### Output

<Properties>
  <Property name="value">
    A 32-byte buffer.
  </Property>
</Properties>

<Row>

<Properties>


  <Property name="input">

  </Property>


</Properties>

<Properties>
  <Property name="value">

  </Property>
</Properties>
