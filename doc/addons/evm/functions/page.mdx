## chain {{ tag: 'function' }}

<Row>
  <Col>

`evm::chain` generates a default chain id and RPC API URL for a valid EVM compatible chain name.

### Inputs

<Properties>


  <Property name="input">
    A EVM-compatible chain name. See https://chainlist.org for a list of supported chains.
  </Property>


</Properties>

### Output

<Properties>
  <Property name="value">

## get_contract_from_foundry_project {{ tag: 'function' }}

<Row>
  <Col>

`evm::get_contract_from_foundry_project` retrieves the deployment artifacts for a contract in a Foundry project.

### Inputs

<Properties>


  <Property name="contract_name">
    The name of the contract being deployed.
  </Property>


  <Property name="contract_filename">
    The `.sol` file that the contract is located in. Defaults to `<ContractName>.sol`.
  </Property>


  <Property name="foundry_manifest_path">
    The location of the Foundry.toml. Defaults to `./foundry.toml`.
  </Property>


  <Property name="foundry_profile">
    The foundry profile that should be used to find the compiled output. Defaults to `default`.
  </Property>


</Properties>

### Output

<Properties>
  <Property name="value">
    Coming Soon
  </Property>
</Properties>

variable "contract" {
    value = evm::get_contract_from_foundry_project("MyContract")
}
output "abi" {
